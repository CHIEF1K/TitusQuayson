# Titus Quayson - Cloud & Site Reliability Engineer Portfolio

A production-grade portfolio website for Cloud, DevOps, and Site Reliability Engineer Titus Quayson.

## Features

- **Dark mode design** with professional, clean aesthetics
- **Production metrics** showcasing 33+ countries supported and 99.9%+ uptime
- **Featured case studies** highlighting GitOps and CI/CD implementations
- **Responsive design** optimized for all devices
- **Fast performance** with Next.js and static export
- **SEO optimized** for technical recruiters and engineering managers

## Tech Stack

- **Next.js 14** - React framework with static export
- **TypeScript** - Type safety and better developer experience
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Clean, consistent icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

This site is configured for static export and can be deployed to any static hosting service:

- Vercel
- Netlify
- GitHub Pages
- AWS S3 + CloudFront
